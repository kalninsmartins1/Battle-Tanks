# Battle-Tanks
Prototy for 3d tank battle game using Unreal engine 4.
## Main game mechanics:
	- Steer tank using left and right triggers of xbox 360 controller.
	- Use right stick to aim.
  - AI openents.
  - Deliver damage to AI oponents.
  - Take damage from AI oponents.
 ## Music & Sound

 - Shoot sound.
 - Tank explode sound.
